Settings

InScien maps your library fully locally. Narration uses a model you choose: local Ollama for privacy, or OpenAI for higher-quality cloud narration.

Profile
Used for local personalization only.
Library
The Zotero data folder InScien reads (read-only).

The folder containing zotero.sqlite and storage/. InScien reads it through a private snapshot and never modifies it. After changing this, re-index your collections.

Model
Choose the model used for narration generation.

Models are read from your running Ollama. Larger models usually produce better narration.
