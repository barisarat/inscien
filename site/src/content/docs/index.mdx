## What it does

<CardGrid stagger>
	<Card title="Map your library">
		A citation graph of your Zotero collection from public OpenAlex data: select papers and see
		what they cite (References) and what cites them (Cited by). Needs no model.
	</Card>
	<Card title="Narrate any paper">
		Turn a paper into a spoken-audio narration you can listen to. A model you choose writes
		the script; a local CPU voice reads it aloud and saves an mp3 you can replay.
	</Card>
	<Card title="Local-first and private">
		Reads your Zotero library read-only through a private snapshot. Nothing leaves your
		machine except, optionally, the text you send to your own cloud model, or public DOI
		lookups when you build the Map.
	</Card>
	<Card title="Read the source">
		Open any paper's original PDF inside the app to read it, without leaving InScien.
	</Card>
</CardGrid>

## Install

InScien runs as a local web app in your own browser - one command, the same on Windows, macOS,
and Linux. All you need is [`uv`](https://docs.astral.sh/uv/): it brings its own Python, and the
app carries everything else (the voice engine and audio are bundled - nothing is installed
system-wide).

```bash
# 1) Install uv once
#    macOS / Linux:
curl -LsSf https://astral.sh/uv/install.sh | sh
#    Windows PowerShell:
#    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

# 2) Run InScien (needs Python 3.12 - pin it so uv fetches the right one)
uvx --python 3.12 inscien              # try it (ephemeral, isolated)
uv tool install --python 3.12 inscien  # or install the `inscien` command, then run: inscien
```

InScien starts a local server, opens your browser, and serves everything from your machine. Then
point it at your Zotero folder in Settings. [Full steps ->](/getting-started/installation/)

## How it works

InScien runs locally and opens in your own browser - no separate app to install, no account. It
rides read-only on your existing [Zotero](https://www.zotero.org) library, with no import and no
copy. The **Map** needs no model; **narration** uses a model you connect (a local Ollama, or your
own OpenAI key).

[Get started](/getting-started/installation/) and point it at your Zotero folder to begin.
